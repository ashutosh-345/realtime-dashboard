# Real-Time Data Analytics Dashboard

## Live Dashboard
 https://realtime-dashboard-jatgkwfcmqd6zfrrcdz5do.streamlit.app/

## Project 3 - Navyan Data Analytics Internship

## Tools Used
- Python
- Streamlit
- Requests
- CoinGecko API (Free)

## Features
- Live Cryptocurrency Prices (Bitcoin, Ethereum, Dogecoin, Cardano, Solana)
- Auto-refresh every 30 seconds
- Interactive sidebar controls
- Live bar chart of prices
- 24 hour price change indicators

## How to Run
1. Install requirements: pip install streamlit requests pandas
2. Run: python -m streamlit run dashboard.py
3. Open browser at http://localhost:8501
